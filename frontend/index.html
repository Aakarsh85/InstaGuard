Account Features

Profile Identity
ⓘ
Does the account have a profile photo? Fake accounts very commonly have no profile picture — one of the strongest fake signals.
ⓘ
Is this account set to private? Private accounts with very few followers can be a suspicious signal.
ⓘ
Does the bio contain a website link? Real influencers and businesses usually have an external URL; bots usually don't.
ⓘ
Is the display name exactly the same as the username? e.g. username "john_doe" and display name "john_doe". This is a mild fake signal.
Activity Stats
ⓘ
Number of people following this account. Fake accounts often have very few or suspiciously round numbers.
ⓘ
Number of accounts this user follows. A very high following count with few followers is a red flag.
ⓘ
Total number of posts. Fake accounts typically have 0 or very few posts.
ⓘ
Character count of the profile bio. Real accounts often have meaningful bios; fake ones usually don't. Count the characters in their bio.
Name Analysis
ⓘ
Number of words in the display name. e.g. "John Smith" = 2, "johnsmith99" = 1. Real people usually have 2+ words.
ⓘ
Proportion of digits in the username (0–1). e.g. "user1234" has 4 digits out of 8 characters = 0.50. Bots often have highly numeric usernames.
ⓘ
Proportion of digits in the full display name (0–1). e.g. "John99" = 2 digits out of 6 chars = 0.33. Real names rarely have numbers.
